## 功能
- ✅支持环境视觉识别
- ✅支持语音交互
- ✅支持语音播放
- ✅支持多国语音
- ✅支持多模态大模型
- ✅支持LVGL9.1
- ✅支持触摸屏交互
- ✅支持屏幕亮度调节。
